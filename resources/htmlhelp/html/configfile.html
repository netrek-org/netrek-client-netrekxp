Configuration File

Any of your program configuration options can be saved in a configuration file. This way, after taking the time to choose your favorite settings, they are automatically loaded when you fire up the client to play another game.

NetrekXP searches for following files (in order from top to bottom) in the executable directory:

  • netrekrc
  • netrekrc.txt
  • xtrekrc
  • xtrekrc.txt
So if you have one of these files it will be read at startup and then change application defaults to whatever values are set in the configuration file.
