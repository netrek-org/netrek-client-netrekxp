General Configuration Options

The following table contains all possible general configuration options and their defaults:
Name Action Possible Values

  • off

  • 1 - white

default: 60

  • on
  • off (default)

default: 0

b2Keymap Keymap for middle mouse button for mouseAsShift mode string

b4Keymap Keymap for fourth mouse button for mouseAsShift mode string
b5Keymap Keymap for fifth mouse button for mouseAsShift mode string
beepOnPrivateMessage Sound beep if received personal message
Doesn't beep for messages from GOD and yourself
  • on
  • off (default)
buttonMap Buttonmap settings for mouse string
buttonMap-<ship>
<ship> can be one of sc,dd,cs,bb,as,sb,ga,att		 Buttonmap settings for specific ship type string
